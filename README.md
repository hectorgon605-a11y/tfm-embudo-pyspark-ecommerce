# TFM - Embudo de Ventas Inteligente con PySpark y Pandas

**Sistema completo de análisis y optimización de ventas para E-commerce usando Big Data**

## 🎯 Objetivo del Proyecto

Desarrollar un **embudo de ventas automatizado** combinado con un potente **toolkit de análisis** basado en PySpark y Pandas para tiendas online.

## ✨ Características Principales

- Procesamiento escalable de datos con **PySpark**
- Análisis RFM avanzado de clientes
- Cálculo de Customer Lifetime Value (CLV)
- Forecasting de ventas
- Comparativa de rendimiento Pandas vs PySpark
- Pipeline ETL completo
- Embudo de ventas automatizado (Lead Magnet + Secuencia de emails)

## 🛠 Tecnologías Utilizadas

- **PySpark** (Big Data)
- **Pandas** (análisis rápido)
- Python 3.10+
- Matplotlib / Plotly
- Git + GitHub

## 📁 Estructura del Proyecto

- `data/` → Datasets (raw, processed, synthetic)
- `notebooks/` → Análisis y experimentación
- `scripts/` → Código en producción
- `src/` → Módulos reutilizables
- `reports/` → Resultados y visualizaciones
